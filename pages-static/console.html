UI v4.3 · Worker 控制台

同域部署：粘贴订阅或 Base64，一键生成 sing-box / Clash 配置。

 

Clash 不区分目标设备，请求不传 device 与 version（由服务默认）。分流/骨架仍生效。若你曾选过 Sing-Box 版本，可在切回 Sing-Box 时继续使用。

用于访问控制台按钮与转换接口；不需要时可留空。若勾选「面板 API 开放到局域网」，此处也会作为面板 API 的密钥（secret）。

上栏选 ACL4SSR 预设（与 /templates 内建一致，传 template=builtin:…）或选 自定义 URL 后填写在线 .ini。选「服务端默认」不传模板参数，由后端使用默认 ACL4SSR 在线分流。Sing-Box 若要用 JSON 模板请走 API。

骨架选项（可选，默认折叠）

按共用 / Sing-Box / Clash 分组。共用里含「调试日志」与 DNS 防泄漏。与分流规则正交。示例 skeleton=log_debug,dns_anti_leak（逗号、空格、|、+ 均可分隔）。数据来自 /profiles 的 skeletons（含 scope、detail）。按当前「生成类型」只显示与 Sing-Box 或 Clash 相关的项；切回另一格式时仍会保留你之前的勾选。全不选则不传 skeleton。
